# dotfiles

My vim/zsh/tmux/powerline/airline dream setup.

To setup:

```
$ git clone https://github.com/paralin/dotfiles.git ~/.janus
$ cd ~/.janus && ./setup_all.sh
```

![](http://i.imgur.com/Bt0vfz7.png)

Dependencies
============

This requires the following:

 - `build-essential`
 - `python-dev`
 - `cmake`
 - `go`
 - `bison`
 - `terminator`
